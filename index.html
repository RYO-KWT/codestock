Main Visual

accordion

accordion

アコーディオンは、クリックすると隠れていたコンテンツが展開されるインターフェース要素です。FAQセクションなどでよく使用されます。 CSSでmax-heightを調整し、JavaScriptでクリックイベントを設定することで開閉を制御できます。適切なaria-expandedとaria-hiddenを付与し、キーボード操作にも対応させるとユーザーにとって使いやすくなります。

dropdown

ドロップダウンメニューは、クリックまたはホバーで開閉するナビゲーション要素の一つです。主にナビゲーションバーやフォームで選択肢を表示するために使用されます。 CSSではvisibilityやopacityを活用してスタイルを制御し、JavaScriptではクリックイベントを利用して開閉の動作を実装します。また、アクセシビリティを考慮し、キーボード操作やWAI-ARIA属性を適用することが推奨されます。

tab

タブは、複数のコンテンツを切り替えて表示するUIコンポーネントです。通常、ボタンやリスト要素でタブを作成し、クリックすると対応するコンテンツを表示します。 JavaScriptでは、is-activeクラスの切り替えを行い、CSSで選択中のタブと非表示のコンテンツを制御します。キーボード操作をサポートするために、aria-selectedやaria-controlsを設定すると、アクセシビリティが向上します。

modal

モーダルは、ポップアップのように画面の中央に表示されるUI要素で、ユーザーの操作を妨げることなく追加情報を表示する際に使用されます。 モーダルの開閉はJavaScriptで制御し、背景のオーバーレイを追加してフォーカスをモーダル内に限定することで、ユーザビリティを向上させます。閉じるボタンやESCキーでのクローズ処理も重要です。

slider

スライダーは、画像やコンテンツを横または縦にスライドして表示するUIコンポーネントです。カルーセルとも呼ばれ、商品の紹介やギャラリーに使われます。 CSSアニメーションやJavaScriptのイベントを使用してスライドを制御し、ユーザーがボタンやスワイプ操作で移動できるように設計します。オートプレイ機能やアクセシビリティ対応を考慮することも重要です。

HTML（HyperText Markup Language）

HTMLは、Webページの構造を定義するためのマークアップ言語です。Webサイトを構築する際の基本となるもので、文書の意味や関係性を示すためのタグを用いて記述します。例えば、見出し、段落、リンク、画像、リストなどの要素を使用し、コンテンツを構築します。 また、HTMLにはフォームやボタンなど、ユーザーとやり取りをするための要素もあります。さらに、セマンティックタグを使うことで、Webページの構造をわかりやすく整理できます。 HTML自体にはデザインの指定機能はなく、基本的にCSSと組み合わせて見た目を整えます。また、JavaScriptを使用することで、動的な処理を加えることも可能です。

CSS（Cascading Style Sheets）

CSSは、HTMLで作成したWebページのデザインやレイアウトを指定するスタイルシート言語です。文字のフォントや色、大きさ、行間、背景の設定、ボックスの余白、要素の配置などを制御できます。 また、CSSにはレスポンシブデザインを実現するための機能もあり、メディアクエリを使って画面サイズごとにスタイルを調整できます。さらに、関数を活用すれば、画面幅に応じてフォントサイズやレイアウトを柔軟に調整できます。 CSSでは、疑似クラスを使用することで、マウス操作やキーボード操作に応じたスタイルの変更も可能です。また、アニメーションを使うことで、スムーズな動きを追加できます。 近年では、CSSだけで実装可能な要素も増えており、ニュースティッカーやアコーディオンなどもCSSのみで動作させることができます。

JavaScript

JavaScriptは、Webページに動的な動作を追加するためのプログラミング言語です。ユーザーの操作に応じたインタラクティブな動作を実装できます。例えば、ボタンをクリックするとメニューが開閉する、フォームに入力されたデータをリアルタイムでチェックする、スクロールに応じてアニメーションを実行するなどの処理が可能です。 JavaScriptはDOMを操作することで、HTML要素を追加・変更・削除できます。例えば、要素を取得し、クラスを切り替えれば、タブメニューやアコーディオン、ドロップダウンメニューの開閉を制御できます。 また、イベントリスナーを使えば、クリックやキー入力、マウスホバーなどの操作に応じて動作を変更できます。キーボード操作に対応する場合は、特定のキーを押した際の挙動を指定できます。 JavaScriptは、WAI-ARIAを活用することでアクセシビリティを向上させることもできます。例えば、アコーディオンの開閉状態を示したり、適切に設定してキーボード操作を可能にしたりすることで、誰でも使いやすいWebサイトを作ることができます。 このように、HTMLがWebページの骨組みを作り、CSSがデザインを整え、JavaScriptが動きを加えることで、より快適なWeb体験を提供できます。
